# Installation

## Requirements

* NVIDIA GPU with CUDA drivers and libraries
* cupy library
* 其他依赖库（比如mrcfile等）

## How to install

### 1 Clone repository

```bash

git clone https://github.com/ZhenHuangLab/MemXTerminator.git

```

### 2 Create a virtual environment

```bash

conda create -n mxt python=3.9

conda activate mxt

```

### 3 Install software and dependencies

```bash

cd MemXTerminator

pip install .

```
